# 📘 Semana 3 – Programación Orientada a Objetos

Proyecto correspondiente a la **actividad de la semana 3** de Programación Orientada a Objetos (POO), desarrollado en Java.

***

## 📌 Descripción

Este proyecto implementa conceptos básicos de **POO en Java**, como:

* Clases y objetos
* Encapsulación
* Herencia
* Composición

Se utilizan entidades como:

* `Persona`
* `Rut`
* `Direccion`
* `Guia`

para modelar un sistema sencillo de gestión de personas.

***

## 🛠️ Tecnologías utilizadas

* Java
* Git
* GitHub

***

## 📂 Estructura del proyecto (ejemplo)

```
src/
 ├── app/
 │   └── Main.java
 ├── model/
 │   ├── Persona.java
 │   ├── Rut.java
 │   ├── Direccion.java
 │   └── Guia.java
```

***

## ▶️ Ejecución

1. Clonar el repositorio:

```bash
git clone https://github.com/BackRob/Semana3.git
```

2. Entrar al proyecto:

```bash
cd Semana3
```

3. Compilar:

```bash
javac Main.java
```

4. Ejecutar:

```bash
java Main
```

***

## 🎯 Objetivo del proyecto

Practicar:

* Modelado de clases en Java
* Validación de datos con excepciones
* Uso de POO en un entorno real
